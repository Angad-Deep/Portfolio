Featured Project



WhatsApp

    Overview-
    This project is a messaging application inspired by Meta's WhatsApp, developed using block-based coding on Scratch. Despite a rigorous study curriculum, I dedicated 2-3 hours each week to this independent project, resulting in a sophisticated communication platform with various functional features.

    Features-
  • Messaging:
    • Three bot chats for interactive communication.
    • Multi-lingual support for sending and receiving messages in multiple languages.
  • Voice and Video Calls:
    • Functionality for audio and video calling.
  • Status Updates:
    • Users can post and view status updates.
  • Communities:
    • Basic community features are included, with plans for full implementation.
  • Customization:
    • Light and dark themes.
    • Pre-set in-chat wallpapers.
  • User Settings:
    • Help settings, privacy settings, and profile settings.
  • Chat Management:
    • Search feature, archiving, pin chats, clear chat option, and block feature for chats.
  • Additional Features:
    • Contact information for bot chats.
    • Multi-device accessibility.
    • Dedicated mobile keyboard and emoji entry options.
    • Latest design scheme for a modern look.
    Technical Details-
  • The project contains approximately 25,000 blocks of code, demonstrating the potential of block-based coding for creating complex applications.
  • Cross-platform accessibility ensures the app can be used on various devices.

    • Conclusion:
    This project effectively showcases how block-based coding can be utilized to develop functional and user-friendly applications.
